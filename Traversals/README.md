## Binary Tree Traversal:

			A
		      /	  \
		    B	    C
		   / \ 
		  D   E


#### Depth First Traversals:
* Inorder (Left, Root, Right) : D B E A C
* Preorder (Root, Left, Right) : A B D E C 
* Postorder (Left, Right, Root) : D E B C A 

#### Breadth First or Level Order Traversal : 
* A B C D E
#### Leaf Nodes:
* B C E
